# IT风险及内控管理平台
## 功能模块与功能点梳理（精简版）

## 1. 建设范围总览
- 一期（可直接开发）：资源管理、计划管控、项目管理、整改管理、OMS/通知/归档集成
- 二期（文档已标注待补充详细设计）：内控工作台、智能内控

## 2. 一期功能模块与功能点

### 2.1 资源管理
#### 2.1.1 标准管理（标准文档）
- 文档全生命周期：上传、审核、发布、更新、失效/激活、逻辑删除
- 文档检索：关键词、分类、标签、状态、上传人、时间范围
- 文档预览与下载：支持最新版本及指定版本
- 版本管理：自动/手动版本号、版本唯一、历史版本永久保留
- 状态流转：草稿 -> 待审核 -> 已发布 -> 失效（可激活）
- 分类管理：树形分类增删改、启用/禁用、删除前关联校验
- 标签管理：新增/编辑/删除、关联校验

#### 2.1.2 控制清单管理
- 模板下载与Excel导入
- 在线编辑清单条目
- 清单版本管理：历史版本、回滚
- 清单状态管理：启用/停用
- 多清单并存：多个启用清单可同时被项目引用
- 项目可引用多清单，并按版本建立引用关系

#### 2.1.3 档案管理
- 项目收尾自动归档（由项目模块触发）
- 档案台账：按项目查询、筛选
- 档案详情：按“项目 -> 清单项 -> 阶段文件”查看
- 文件操作：在线预览、单文件下载、整项目ZIP打包下载

#### 2.1.4 人员管理
- 人员台账：新增、编辑、启用/禁用（在职/离职）、导出
- 角色管理：角色增删改、启停、权限配置入口
- 标签管理：标签增删改、引用校验
- 人员选择服务：供计划/项目/整改模块按部门、角色、标签筛选

### 2.2 计划管控
- 计划体系：主计划/子计划树（年度/季度/月度）
- 计划编制：新增、编辑、父子层级维护
- 状态管理：草稿、已发布、执行中、已完成、已取消
- 列表与详情：树形展示、筛选查询、详情查看
- 提醒配置：提前提醒、延期提醒、通知方式（邮件/企业微信）
- 变更记录：状态变更与信息修改日志

### 2.3 项目管理
- 项目启动：从计划生成/手动创建
- 清单任务生成：全量核查、按期核查、随机抽取、手动增删
- 团队管理：项目经理、执行人、评审人维护
- 任务流转：待启动 -> 待分发 -> 待提交 -> 待评审 -> 已完成/整改中/退回
- 任务处理：分发、上传资料、提交、评审（通过/退回）
- 整改联动：评审退回可生成整改单，整改完成后任务回写完成
- 项目文档管理：上传、下载、删除
- 统计看板：完成率、整改率、趋势、分布、处理时长
- OMS集成：任务同步记录、失败重试
- 项目收尾：报告生成、PDF导出、触发归档

### 2.4 整改管理
- 整改单来源：项目任务触发/手动新建
- 整改单流转：待整改 -> 待审核 -> 已完成（或退回后重整）
- 列表管理：查询筛选、查看详情、编辑、删除（按状态权限）
- 整改执行：上传整改资料、提交整改
- 审核处理：通过/退回、审核日志
- 项目联动：整改完成回写项目任务状态
- OMS联动：同步至OMS、同步状态展示、手动重试、回调更新

## 3. 二期预留模块（当前文档未展开详细设计）

### 3.1 内控工作台
- 驾驶舱（项目进展、风险态势、待办聚合）
- 告警中心
- 日志中心

### 3.2 智能内控
- 规则库/模型库/工具库
- OCR/RPA/大模型能力接入
- 跨模块数据分析与智能推荐

## 4. 关键跨模块流程（开发时必须打通）
- 计划 -> 项目：项目可引用计划数据生成
- 资源（清单/人员） -> 项目：项目创建与组队依赖基础资源
- 项目 -> 整改：评审退回时生成整改单
- 整改 -> 项目：整改完成回写任务状态
- 项目 -> 档案：项目收尾后自动归档
- 项目/整改 -> OMS：任务与整改单跨系统同步

## 5. 一期开发优先级建议
1. P0：资源管理（标准文档+清单+人员）
2. P0：项目管理（含任务流转）
3. P0：整改管理（含项目联动）
4. P1：计划管控（支撑项目计划化启动）
5. P1：OMS同步、通知提醒、归档打包下载
6. P2：工作台与智能内控（二期）
